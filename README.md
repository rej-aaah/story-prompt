# Story Prompt Generator for Kids

This is a simple Python program that generates creative story prompts for children.  
The goal of this project is to encourage imagination and storytelling using basic programming concepts.

## How it Works

The program randomly selects:
- A character
- A setting
- A challenge

It then combines them to generate a unique story idea each time the program runs.

Example output:

Story Idea: a curious turtle in a magical forest who discovers a mysterious map.

## Technologies Used

- Python
- Random module

## Purpose

This project explores how simple algorithms can support creativity and storytelling for kids.  
It can be extended into storytelling tools or interactive educational devices.
